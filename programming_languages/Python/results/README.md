### Python results

Contains the reports for each of the techniques in the following format:
- **Technique_Test1** (using CPython interpreter without applying the technique)
- **Technique_Test2** (using CPython interpreter after applying the technique)
- **Technique_Test1_Nuitka** (using Nuitka compiler without applying the technique)
- **Technique_Test2_Nuitka** (using Nuitka compiler after applying the technique)

In addition to the results mentioned above, there is a series of Excel files with the global results of the test cases. A folder named `images` contains visual representations of the data.
