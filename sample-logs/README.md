## Sample Logs Folder

This folder contains example log files illustrating the raw data generated by the EET measurement instrument.

The JSON file includes measurement information for the corresponding entity class.

Each line in **4AlarcosComputaex_T4_3088_EX** contains values corresponding to the following fields: `id`, `time`, `monitor`, and `DUT` (Device Under Test).

Each line in **4AlarcosComputaex_T4_3088_IN** contains values corresponding to the following fields: `id`, `time`, `HDD(1)`, `HDD(2)`, `Graphics card(1)`, `Graphics card(2)`, `Processor(1)`, and `Processor(2)`.

The DUT hardware components are equipped with two sensors; therefore, two values are recorded per component. The final DUT value is computed as the sum of these two sensor readings.

**Note:** The remainder of the logs from this study are not included in this repository for practical reasons due to their large size.
